Business Logic URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/:title GET None A JSON object holding data about a single movie, containing an ID, realesed, description, genre, director, and image URL properties. For example:
"id": 1,
"title": "The lakehouse",
"released": 2006,
"description": "something something",
"genre": "something something",
"director": "something something",
"image URL": "something something"
Get a definition of the genre of a movie /movies/genre/:genreName GET None A text document that defines the genre of the movie
Get data about the director of a movie movies/directors/:directorName/ GET None A JSON object holding data about the director of a movie
User can register /users Post A JSON object holding data about the user to register. For example:
"email": "some email"; "username": "some name"; "password": "some password"; 		A text confirming that the user has been registered
User can update their information /users/:name Post None A text confirming that the user information has been updated
User can add a movie to their list of favourite movies /users/:id/:movieTitle Post None A text confirming that the movie has been added to the favourite movies
User can delete a movie from their list of favourite movies /users/:id/:movieTitle Delete None A text confirming that the movie has been deleted from the favorite movies
User can deregister /users/:id Delete None A text confirming that the user has been deregistered
